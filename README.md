# RaceBox EzzTech v.2.7.0

RaceBox EzzTech adalah aplikasi GPS drag timer untuk test performa kendaraan.

## Developer
Powered By : faizyh

## Fitur Aplikasi
- GPS speed realtime dari sensor lokasi HP
- Test akselerasi:
  - 0–30 km/h
  - 0–60 km/h
  - 0–100 km/h
- Test jarak:
  - 201 meter
  - 402 meter
- Top speed
- Jarak tempuh
- Akurasi GPS
- History run
- Best performance
- Export CSV
- Share result
- Reaction timer
- Launch analyzer
- GPS trace

## Branding
- Nama aplikasi: RaceBox EzzTech
- Header menggunakan logo EzzTech
- Splash screen premium
- Adaptive icon Android
- Powered By : faizyh

## Catatan Instalasi
APK ini didistribusikan manual di luar Play Store.  
Aktifkan izin “Install unknown apps / Instal aplikasi tidak dikenal” jika Android meminta izin saat instalasi.

## Catatan Penggunaan
Gunakan aplikasi di area terbuka agar GPS lebih akurat.  
Pastikan HP stabil dan GPS aktif sebelum melakukan test run.
